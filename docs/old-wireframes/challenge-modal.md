## Key Features:

**Simplified Doubles Flow**
- Select 3 players (not specific partners)
- Single invitation sent to all players
- Teams decided organically when players meet
- No complex partner selection process

**Clear Visual Hierarchy**
- Checkboxes show selected players clearly
- Explanatory text about flexible team arrangement
- Consistent with singles challenge pattern

**Smart Defaults**
- Singles selected by default (most common)
- Tomorrow as default timing
- Pre-filled encouraging message

**Notification Design**
- Clear who initiated the challenge
- Shows all players involved
- Simple accept/decline options
- Includes timing and message context
